# DONIA Social

Plateforme intelligente modulaire dédiée aux services sociaux,
intégrée à l’écosystème DONIA (Éducation • Médical • Social).

## Stack
- Frontend : React + Vite + Tailwind
- Backend : Node.js + Supabase
- Docker ready

## Lancer en local (1 commande)

docker compose up -d --build

Service URLs:
- Frontend : http://localhost:8080
- Backend  : http://localhost:4000

## Build frontend

cd frontend
npm run build
